Sentiment Analysis using CNN and GloVe Embeddings

Overview

This project performs sentiment analysis on movie reviews using a Convolutional Neural Network (CNN) and pre-trained GloVe word embeddings.

Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Pandas
* NLTK
* SpaCy
* Scikit-Learn

Project Structure

* notebook/
* models/
* images/
* predict.py
* configuration.py
